# Fichiers publics

Les fichiers de ce dossier sont servis à la racine du site (ex. `public/logo.png`
est accessible à `/logo.png`).

## Logo du club

Dépose le logo du Centre Sportif de Haute-Performance ici sous le nom **`logo.png`**
(ou `logo.jpg`). Il sera automatiquement :

- intégré en haut des **reçus PDF** ;
- affiché dans les **courriels** (rappels, reçus, bienvenue) — nécessite que la
  variable `APP_URL` pointe vers l'URL publique de l'application ;
- affiché en en-tête du **formulaire d'inscription** en ligne.

Format recommandé : PNG carré, ~512×512 px, fond transparent ou blanc.
Si aucun logo n'est présent, l'application fonctionne normalement, sans logo.
